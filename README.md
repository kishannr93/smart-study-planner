# 🎓 Smart Study Planner & Career Assistant AI

## 📌 Overview
Smart Study Planner & Career Assistant AI is an intelligent web application designed to help students plan their studies efficiently and choose the right career path. It uses AI to analyze user input like interests, goals, and skills to generate personalized study plans and career recommendations.

---

## 🚀 Key Features
- 📅 Personalized Study Planner based on goals
- 🎯 AI-powered Career Guidance
- 🤖 Smart Recommendations using AI
- 📊 User-friendly interface for easy navigation
- 🧠 Helps reduce confusion in career decisions

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: ( Node.js / Python)  
- AI Integration: Google AI Studio (Gemini API)

---

## 💡 How It Works
1. User enters their interests, subjects, or goals  
2. AI processes the input  
3. Generates a personalized study plan  
4. Suggests suitable career paths  
5. Helps user stay organized and focused  

---

## 🎯 Problem Statement
Many students struggle with planning their studies and choosing the right career due to lack of guidance. This app solves that by providing AI-driven personalized solutions.

---

## 🌟 Solution
This application acts as a smart assistant that guides students step-by-step in both academic planning and career decision-making.
